# AI 댓글 프로필 이미지 넣는 곳

이 폴더에 AI 댓글에서 사용할 프로필 이미지를 6개 넣어 주세요.

## 파일 이름

반드시 아래 이름으로 저장합니다.

```text
profile-1.webp
profile-2.webp
profile-3.webp
profile-4.webp
profile-5.webp
profile-6.webp
```

## 권장 형식

- 형식: WebP 권장
- 크기: 정사각형 128×128px 권장
- 최소: 96×96px 이상
- 최대: 256×256px 정도면 충분
- 비율: 1:1 정사각형
- 스타일: 서로 다른 인상으로 6개 준비하면 댓글창이 덜 겹쳐 보입니다.

## 동작 방식

- AI 댓글이 생성될 때 댓글마다 `profile-1.webp`부터 `profile-6.webp` 중 하나를 자동 배정합니다.
- 한 스레드 안에서는 가능하면 같은 이미지가 바로 겹치지 않도록 순서가 섞여 배정됩니다.
- 이미지 파일이 없거나 로딩에 실패하면 기존처럼 닉네임 첫 글자 아이콘으로 표시됩니다.

## PNG/JPG를 쓰고 싶은 경우

현재 앱 코드는 `.webp` 이름을 기준으로 찾습니다. PNG나 JPG를 쓰려면 변환해서 위 이름의 `.webp` 파일로 저장하는 것을 추천합니다.
