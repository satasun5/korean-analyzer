# 반짝국어 — AI 지문 분석기

국어 지문을 넣으면 AI가 지문을 정돈하고, 구조를 분석하고, 형광펜으로 핵심 구절을 표시하고, 마인드맵과 문제 풀이, 메모, AI 댓글까지 제공하는 학습용 웹앱입니다.

## 핵심 기능

### 1. 지문 입력과 정돈

- OCR로 가져온 지문을 붙여넣을 수 있습니다.
- 문단 사이에 빈 줄을 넣으면 문단 기준으로 분석합니다.
- `OCR 정리`는 문단 내부의 강제 줄바꿈을 이어 붙입니다.
- `AI 정돈`은 메뉴의 저비용 모델을 사용해 띄어쓰기, 붙어쓰기, 어색한 줄바꿈을 정리합니다.
- 정돈은 원문 의미를 바꾸지 않는 방향으로 작동합니다.

### 2. AI 분석

- 전체 주제와 분야를 뽑습니다.
- 전체 요약과 읽는 법을 제공합니다.
- 문단별 요약을 생성합니다.
- 글의 구조를 타임라인으로 정리합니다.
- 비교·대조 관계를 정리합니다.
- 개념 사전을 만듭니다.
- 헷갈리는 문장을 쉬운 말로 다시 설명합니다.
- 난이도를 5개 기준으로 나누어 평가합니다.

### 3. 형광펜

- 핵심 주장, 근거, 비교·대조, 개념 정의, 예시, 주의·반론, 부연·균형을 색상별로 표시합니다.
- 형광펜은 어려운 부분만 표시하는 기능이 아니라, 지문을 읽는 데 필요한 핵심 구절을 남기는 기능입니다.
- 같은 구절에 여러 의미가 겹칠 수 있습니다.
- 형광펜에 마우스를 올리면 해당 구절이 무엇과 연결되는지 설명합니다.
- 클릭하면 자세한 설명을 볼 수 있습니다.

### 4. 마인드맵

- 중심 개념에서 여러 가지가 뻗는 방식으로 지문을 시각화합니다.
- 노드를 드래그할 수 있습니다.
- 빈 공간 드래그와 휠클릭 드래그로 화면을 이동할 수 있습니다.
- 마우스 휠과 버튼으로 확대·축소할 수 있습니다.
- 노드를 클릭하면 세부 설명을 볼 수 있습니다.

### 5. 문제 제작

- 문제 제작은 `문제` 탭 안에서 실행합니다.
- 5지선다, OX, 서술형 문제를 생성합니다.
- 문제는 먼저 사용자가 풀고, 채점 후 정답과 해설을 확인하는 방식입니다.
- 5지선다는 정답 선지만 튀지 않도록 선지 밀도와 함정 설계를 강화했습니다.
- OX는 개별 채점과 전체 채점을 지원합니다.
- 서술형은 개별 AI 채점과 전체 AI 채점을 지원합니다.
- 서술형 채점은 메뉴의 저비용 모델을 사용합니다.

### 6. 문제별 질문

- 각 문제에서 납득이 안 되는 선지나 개념을 AI에게 질문할 수 있습니다.
- 정답을 바로 알려주는 것보다, 왜 그 선지가 맞거나 틀리는지 설명하는 데 초점을 둡니다.

### 7. 메모

- 지문에서 이해가 안 되는 구절을 드래그해 메모 탭에서 설명을 요청할 수 있습니다.
- 메모 답변에 다시 이어 질문할 수 있습니다.
- 메모 답변은 Markdown 문법을 보기 좋게 렌더링합니다.
- 답변이 길어져도 카드 안에서 정리되도록 구성했습니다.

### 8. AI 댓글

- `AI 댓글` 탭에서 지문에 대해 질문을 남기면 여러 댓글이 달립니다.
- 댓글은 메뉴의 저비용 모델을 사용합니다.
- 댓글들은 해설 카드가 아니라 실제 댓글창처럼 짧은 반응, 근거 있는 설명, 가벼운 태클, 귀여운 딴소리를 섞어 답합니다.
- `AI` 배지, 성격 라벨, 별도 근거 줄은 표시하지 않습니다.
- 댓글에 다시 질문하면 그 댓글 아래에 답글이 달립니다.
- 지문 근거를 벗어난 거짓 설명은 하지 않도록 제한되어 있습니다.

### 9. 저장

- 분석 결과를 저장하고 다시 열 수 있습니다.
- 저장에는 지문, 분석 결과, 문제, 메모, AI 댓글 상태가 포함됩니다.
- 샘플을 보더라도 내 지문 분석 상태는 유지됩니다.
- `새 분석 노트`로 새 작업을 시작할 수 있습니다.

### 10. 오류 처리

- 일반 안내는 3초 후 사라집니다.
- 오류 메시지는 자동으로 사라지지 않습니다.
- 오류 로그를 열어 원인을 확인할 수 있습니다.

## 실행 방법

프로젝트 폴더에서 로컬 서버를 켭니다.

```bash
py -m http.server 8123
```

macOS/Linux에서는 다음 명령을 사용할 수 있습니다.

```bash
python3 -m http.server 8123
```

브라우저에서 접속합니다.

```text
http://127.0.0.1:8123
```

기존 버전을 열었던 적이 있으면 `Ctrl + F5`로 강력 새로고침하세요.

## GitHub Pages 배포

이 저장소는 정적 웹앱입니다. GitHub Pages로 배포할 수 있습니다.

Settings → Pages → Build and deployment → Source → GitHub Actions
