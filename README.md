# 반짝국어 — AI 지문 분석기

> v3.0.3: 메모 내보내기 레이아웃과 모바일 PDF 인쇄 흐름을 보강했습니다.

국어 지문을 넣으면 AI가 지문을 정돈하고, 구조를 분석하고, 형광펜으로 핵심 구절을 표시하고, 마인드맵과 문제 풀이, 메모, AI 댓글까지 제공하는 학습용 웹앱입니다.

## 핵심 기능

### 1. 지문 입력과 정돈

- OCR로 가져온 지문을 붙여넣을 수 있습니다.
- 문단 사이에 빈 줄을 넣으면 문단 기준으로 분석합니다.
- `OCR 정리`는 문단 내부의 강제 줄바꿈을 이어 붙입니다.
- `AI 정돈`은 메뉴의 저비용 모델을 사용해 띄어쓰기, 붙어쓰기, 어색한 줄바꿈을 정리합니다.
- 정돈은 원문 의미를 바꾸지 않는 방향으로 작동합니다.

### 2. AI 분석

- 전체 주제와 분야를 뽑습니다.
- 전체 요약과 읽는 법을 제공합니다.
- 문단별 요약을 생성합니다.
- 글의 구조를 타임라인으로 정리합니다.
- 비교·대조 관계를 정리합니다.
- 개념 사전을 만듭니다.
- 헷갈리는 문장을 쉬운 말로 다시 설명합니다.
- 난이도를 5개 기준으로 나누어 평가합니다.

### 3. 형광펜

- 핵심 주장, 근거, 비교·대조, 개념 정의, 예시, 주의·반론, 부연·균형을 색상별로 표시합니다.
- 형광펜은 어려운 부분만 표시하는 기능이 아니라, 지문을 읽는 데 필요한 핵심 구절을 남기는 기능입니다.
- 같은 구절에 여러 의미가 겹칠 수 있습니다.
- 형광펜에 마우스를 올리면 해당 구절이 무엇과 연결되는지 설명합니다.
- 클릭하면 자세한 설명을 볼 수 있습니다.

### 4. 마인드맵

- 중심 개념에서 여러 가지가 뻗는 방식으로 지문을 시각화합니다.
- 노드를 드래그할 수 있습니다.
- 빈 공간 드래그와 휠클릭 드래그로 화면을 이동할 수 있습니다.
- 마우스 휠과 버튼으로 확대·축소할 수 있습니다.
- 노드를 클릭하면 세부 설명을 볼 수 있습니다.

### 5. 문제 제작

- 문제 제작은 `문제` 탭 안에서 실행합니다.
- 기본 설정에서 난이도 `상·중·하`, 문제 수 `많음·보통·적음`을 고를 수 있습니다.
- 고급 설정을 열면 5지선다, OX, 서술형별로 문제 개수, 난이도, 길이를 슬라이더로 조절할 수 있습니다.
- 5지선다는 최대 12문항, OX는 최대 30개, 서술형은 최대 10문항으로 제한합니다.
- 기본 방식은 5지선다와 OX·서술형을 두 번의 호출로 나누어 JSON이 길어져 끊기는 문제를 줄입니다.
- `추론 호출 한 번만 사용`을 켜면 비용과 시간을 줄일 수 있지만, 난이도 상과 문제 수 많음은 비활성화됩니다.
- 문제는 먼저 사용자가 풀고, 채점 후 정답과 해설을 확인하는 방식입니다.
- 5지선다는 정답 선지만 튀지 않도록 선지 밀도와 함정 설계를 강화했습니다.
- OX는 개별 채점과 전체 채점을 지원합니다.
- 서술형은 개별 AI 채점과 전체 AI 채점을 지원합니다.
- 서술형 채점은 메뉴의 저비용 모델을 사용합니다.

### 6. 문제별 질문

- 각 문제에서 납득이 안 되는 선지나 개념을 AI에게 질문할 수 있습니다.
- 정답을 바로 알려주는 것보다, 왜 그 선지가 맞거나 틀리는지 설명하는 데 초점을 둡니다.

### 7. 메모

- 분석이 끝나면 본문 위의 `AI에게 질문하기` 카드가 펼쳐진 상태로 표시됩니다. 필요하면 접기 버튼으로 접을 수 있습니다.
- 지문에서 이해가 안 되는 구절을 드래그해 메모 탭에서 설명을 요청할 수 있습니다.
- 선택 구절로 만든 메모는 구절을 보라색 박스에 크게 보여 주고, 바로 아래에 질문을 표시합니다.
- 전체 지문 질문 메모는 기존처럼 질문 중심 카드로 표시합니다.
- 메모 답변에 다시 이어 질문할 수 있습니다.
- 필요 없는 메모는 삭제 버튼으로 제거할 수 있습니다.
- 메모 답변은 Markdown 문법을 보기 좋게 렌더링합니다.
- 답변이 길어져도 카드 안에서 정리되도록 구성했습니다.

### 8. AI 댓글

- `AI 댓글` 탭에서 지문에 대해 질문을 남기면 여러 댓글이 달립니다.
- 댓글은 메뉴의 저비용 모델을 사용합니다.
- 댓글들은 해설 카드가 아니라 실제 댓글창처럼 짧은 반응, 근거 있는 설명, 가벼운 태클, 귀여운 딴소리를 섞어 답합니다.
- `AI` 배지, 성격 라벨, 별도 근거 줄은 표시하지 않습니다.
- 댓글에 다시 질문하면 그 댓글 아래에 답글이 달립니다.
- `assets/chat-profiles/profile-1.webp`부터 `profile-6.webp`까지 넣으면 댓글마다 프로필 이미지가 자동으로 다양하게 배정됩니다.
- 프로필 이미지가 없으면 기존처럼 닉네임 첫 글자 아이콘으로 표시됩니다.
- 지문 근거를 벗어난 거짓 설명은 하지 않도록 제한되어 있습니다.

### 9. 내보내기

- 분석 완료 후 상단의 `내보내기` 버튼으로 결과를 파일로 저장할 수 있습니다.
- PDF, Markdown, TXT 중 하나를 선택할 수 있으며 PDF가 기본 추천 형식입니다.
- PDF는 A4 인쇄용 레이아웃으로 브라우저의 `PDF로 저장` 기능을 사용합니다. 데스크톱에서는 숨겨진 인쇄 프레임을, 모바일에서는 인쇄 전용 탭을 열어 현재 웹사이트 화면이 그대로 저장되는 문제를 줄였습니다.
- 내보낼 항목을 체크박스로 고를 수 있습니다: 본문, 본문 형광펜·주석, 요약, 구조, 개념, 문제, 해설, 메모, AI 댓글.
- 본문 형광펜을 포함하면 문단 본문 바로 아래에 해당 문단의 형광펜 설명 카드가 4열로 배치됩니다.
- 문제를 포함할 때 해설 포함을 선택하면 정답, 출제 의도, 선지 해설, 서술형 모범 답안이 마지막 해설 페이지에 따로 모입니다.
- 메모 내보내기에서는 선택 구절 질문을 사이트와 동일하게 보라색 선택 구절 박스와 `ㄴ 질문` 구조로 표시합니다. Markdown/TXT에도 선택 구절과 질문 관계를 함께 담습니다.
- Markdown은 다른 AI 어시스트나 공유용으로 원문과 분석 맥락을 충분히 담습니다.
- TXT는 가볍게 복사하거나 보관하기 위한 순수 텍스트 형식입니다.

### 10. 저장

- 분석 결과를 저장하고 다시 열 수 있습니다.
- 저장에는 지문, 분석 결과, 문제, 메모, AI 댓글 상태가 포함됩니다.
- 샘플을 보더라도 내 지문 분석 상태는 유지됩니다.
- `새 분석 노트`로 새 작업을 시작할 수 있습니다.

### 11. 오류 처리

- 일반 안내는 3초 후 사라집니다.
- 오류 메시지는 자동으로 사라지지 않습니다.
- 오류 로그를 열어 원인을 확인할 수 있습니다.
- JSON 파싱 실패 시 스키마 이름, 사용 모델, 응답 길이, 오류 위치, 오류 위치 주변 텍스트, 응답 마지막 부분, 토큰 사용량을 함께 기록합니다.

## 실행 방법

프로젝트 폴더에서 로컬 서버를 켭니다.

```bash
py -m http.server 8123
```

macOS/Linux에서는 다음 명령을 사용할 수 있습니다.

```bash
python3 -m http.server 8123
```

브라우저에서 접속합니다.

```text
http://127.0.0.1:8123
```

기존 버전을 열었던 적이 있으면 `Ctrl + F5`로 강력 새로고침하세요.

## 버전

현재 버전: **v3.0.1**

변경 내용:

- 문제 제작 설정을 추가했습니다.
- 기본 설정에서 난이도 상·중·하와 문제 수 많음·보통·적음을 선택할 수 있습니다.
- 고급 설정에서 5지선다, OX, 서술형별 문제 수·난이도·길이를 슬라이더로 조절할 수 있습니다.
- 문제 제작을 기본적으로 5지선다 호출과 OX·서술형 호출로 나누어 JSON 출력이 너무 길어지는 문제를 줄였습니다.
- 비용 절약을 위한 `추론 호출 한 번만 사용` 옵션을 추가했습니다.
- 문제 제작 버튼 스타일을 주변 UI와 어울리도록 낮췄습니다.
- AI 질문 카드는 분석 완료 후 펼쳐진 상태로 보이며, 사용자가 접거나 펼칠 수 있습니다.
- JSON 파싱 실패 로그를 정밀화했습니다. 오류 위치 주변, 응답 마지막 부분, 스키마명, 모델, 사용량을 함께 기록합니다.
- PDF 내보내기 관련 인쇄 흐름 수정은 유지됩니다.
- PDF 페이지 번호가 0으로 고정되던 표시를 제거하고, 인쇄용 페이지 번호를 중앙 정렬 방식으로 정리했습니다.
- PDF 본문/형광펜 주석 영역을 7:3 연속 레이아웃으로 압축해 본문 중간 공백을 줄였습니다.
- PDF 메모 섹션에 질문 대상 구절과 강조 표시를 반영했습니다.
- PDF 문제 해설 섹션을 카드형에서 압축된 텍스트 목록형으로 바꿨습니다.
