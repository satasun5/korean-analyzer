# 스트리밍 기능 안내

현재 안정판(v2.0)은 분석 결과를 JSON Schema에 맞게 한 번에 받아 화면에 반영합니다.
국어 분석기는 형광펜 위치, 문단 ID, 마인드맵 노드, 문제 데이터가 서로 연결되어 있어서,
중간 스트리밍 텍스트를 곧바로 UI에 반영하면 JSON 파싱 오류나 위치 매칭 오류가 쉽게 발생합니다.

그래서 스트리밍은 `streaming-analysis.js`에 실험용 함수로 분리했습니다.
나중에 구현할 때 권장 흐름은 다음과 같습니다.

1. 스트리밍으로 “현재 단계 안내/간이 요약”만 먼저 보여주기
2. 최종 JSON은 별도 비스트리밍 요청으로 받아 검증하기
3. 검증된 JSON만 본문 형광펜, 마인드맵, 문제 UI에 반영하기

즉, 스트리밍은 화면 체감 속도 개선용으로 쓰고,
최종 데이터는 안정적인 Structured Output으로 받는 구조가 안전합니다.
